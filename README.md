# Задание 3. Используя «1С:Система проектирования прикладных решений», зафиксируйте группирующую функцию «Оптовые продажи клиентам», а в её составе распишите следующие Функции системы:
* Приём заказа клиента
* Выставление счёта клиенту
* Фиксация оплаты по счёту
* Подбор товара на складе
* Реализация и доставка товара клиенту
Представьте схему группирующей функции.
Для каждой из указанных Функций системы проработайте состав объектов метаданных.
### Требования к программной архитектуре
Должны быть представлены следующие объекты конфигурации:
* Справочники
* Документы
* Регистры накопления (Регистры сведений — по необходимости).
Для каждого объекта должен быть представлен пример реквизитного состава, включая реквизиты табличных частей.
Для регистров накопления определите Измерения и Ресурсы; при этом Измерения должны иметь ссылочный тип данных.
Для ряда реквизитов (в справочниках и документах) определите ссылочный тип данных.
